### about *.map

the basic usage about *.map file

#### direction 

to show the game scene
1. the scope is a png image such as sea/sky/mountain..., 
2. bgLayer is shown [grass, ground...]
3. the role, npc, box, transfer, etc...
4. npcLayer [tree, etc...]
